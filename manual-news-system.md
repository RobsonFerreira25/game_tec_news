# Plano: Sistema de Atualização de Notícias Manual

Este documento descreve as etapas para permitir que você atualize o site apenas fornecendo os links das notícias.

## Objetivo
Facilitar a atualização de conteúdos sem precisar mexer no código React.

## O que vamos fazer:
1. **Dinamicidade**: Transformar as notícias fixas em um arquivo de dados (`news.json`).
2. **Automação**: Criar um script chamado `update_news.py` que lê seus links e gera o conteúdo automaticamente.
3. **Facilidade**: Você só precisará colocar o link e rodar um comando.

## Próximos Passos (Tarefas):
- Criar a pasta de dados e o arquivo JSON.
- Alterar o site para ler esse arquivo.
- Desenvolver o script de automação inteligente.
